# Hogwarts Diary – Flask Web Application

A **Hogwarts-themed personal diary web application** built using **Python Flask** and **MySQL**.
This project allows users to create, view, edit, and delete diary entries through a magical Hogwarts-inspired interface.

The main objective of this project was to practice **backend development using Flask**, integrate a **MySQL database**, and build a simple full-stack web application implementing CRUD operations.

---

## Features

* Create new diary entries
* View saved diary entries
* Edit existing entries
* Delete unwanted entries
* Hogwarts-themed user interface
* Data stored and managed using MySQL database

## Installation and Setup

Clone the repository

```
git clone https://github.com/your-username/hogwarts-diary-flask-webapp.git
```

Navigate to the project directory

```
cd hogwarts-diary-flask-webapp
```

Create and activate a virtual environment

```
python -m venv venv
venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

Configure MySQL database and update database credentials in `app.py`.

Run the application

```
python app.py
```

Open the application in your browser

```
http://127.0.0.1:5000
```
